Cálculo do churrasco

🏠 📝 👥

A hora de fazer a conta de quantidades para um churrasco pode ser difícil, esta calculadora pode ajudar na hora de programar seu churrasco.

Utilizada para saber o quanto de carne, cerveja e carvão que são necessários para um churrasco. Abaixo você terá duas opções, calcular baseado no número de pessoas ou fazer uma lista com os nomes para saber a quantidade.

Este cálculo leva em consideração uma média nacional, podendo variar conforme a necessidade, utilize como uma base para suas compras.
